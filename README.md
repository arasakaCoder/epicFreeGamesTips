# epicFreeGamesTips

免费游戏的提醒，是提醒不是自动领取


我本人就是经常忘记领游戏，于是简单写了个获取免费游戏信息的小爬虫。

## 使用方式

有Python环境直接运行即可
或者用pyinstaller把它打包成独立的可执行文件，
```bash
pyinstaller -F .\findEpicFreeGames.py
```
然后设置为每天定时启动。
